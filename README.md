# Congregacional Theme #
* [Real Demo](http://congregacional.org.br/)
* [GitHub Author](https://github.com/isaquemelo)


## Installation ##

Send the folder "congregacional" to the default themes folder from WordPress `wp-content/themes` or send the zipball using the WordPress Theme Installer.

## Documentation ##

Informations about how-to use the classes and functions from Odin is on the [Official Wiki](https://github.com/wpbrasil/odin/wiki/)
